# 技术指导委员会（TSC-Technical Steering Committee）定位与职责

- 定义BoostKit项目的技术发展方向和愿景，决策重大技术事项；
- 制定、修改技术指导委员会相关制度；
- 选举、任免技术指导委员会主席及委员;
- 决策SIG（Special Interest Group 特别兴趣小组）的成立和撤销；
- 其他对社区有重要影响的技术工作。

## 会议准则  

会议频次：常规会议跟随版本会议开展，欢迎订阅[邮箱]( releasemanagment@lists.boostkit.osinfra.cn)了解相关会议公告及内容。

## 会议纪要  

您可以在[会议纪要](https://etherpad.boostkit.osinfra.cn/p/ReleaseManagment)查看到相关内容，TSC并会在每次会议后将会议纪要发布到releasemanagment@lists.boostkit.osinfra.cn邮件列表中。
